Concept icône

Un cadre polaroid avec un soleil à l'intérieur. Le cadre = "frame". Le soleil = "mood". Simple, lisible à 16px, mémorable.

05 · Règles d'usage

✅ À faire

Toujours utiliser Cormorant Garamond pour les titres. Gold sur fond sombre uniquement. Logo avec espace minimum autour. Tagline en lowercase.

❌ À éviter

Jamais de fond clair avec texte gold (trop peu de contraste). Jamais de logo déformé. Pas d'autre police que les 3 définies. Pas de gold sur fond blanc.

📱 Mobile

Fond toujours sombre (#0A0906 ou #1A1810). Texte cream (#F0E8D0). Accents gold uniquement pour les éléments clés — bouton principal, logo, highlights.

🌐 Web / Landing

Version gradient du logo pour le hero. Fond dégradé radial avec nuances chaudes. Garder le ton premium — peu d'éléments, beaucoup d'espace.
